## 1. 系统简介

### 1.1 系统概述
京东仓储管理系统（WMS）是一套面向仓储企业的综合性管理平台，支持入库管理、出库管理、库存监控、任务调度、绩效管理等核心功能。

### 1.2 角色说明
| 角色 | 描述 | 主要功能 |
|------|------|----------|
| 操作员 | 负责日常仓储作业 | 入库、出库、任务处理、货位管理、库存盘点、异常处理 |
| 仓储主管 | 负责运营监控管理 | 库存监控、任务调度、绩效管理、质量检查、预警管理 |
| 管理员 | 负责系统配置维护 | 用户管理、角色管理、仓库管理、数据备份、操作日志 |

### 1.3 登录系统
**截图位置**：暂无（请参照登录页面）

**操作步骤**：
1. 打开浏览器，访问系统地址
2. 选择登录角色（操作员/仓储主管/管理员）
3. 输入用户名/工号（或手机号）和密码
4. 点击"登录"按钮进入系统

## 2. 操作员使用手册

### 2.1 入库管理

#### 2.1.1 查看入库任务
**截图位置**：暂无

**操作步骤**：
1. 登录系统后，点击左侧菜单"入库管理"
2. 系统展示入库单列表，默认显示当前操作员的所有入库单
3. 支持按状态筛选（待入库/已完成）

#### 2.1.2 创建入库单
**截图位置**：暂无

**操作步骤**：
1. 在入库管理页面，点击"创建入库单"按钮
2. 弹出创建入库单表单
3. 选择商品（系统支持模糊搜索）
4. 输入批次号
5. 选择或系统推荐空闲货位
6. 输入入库数量
7. 点击"提交"按钮

#### 2.1.3 完成入库
**截图位置**：暂无

**操作步骤**：
1. 在入库单列表中，找到需要完成的入库单
2. 点击"完成入库"按钮
3. 系统自动更新库存，货位状态变为"占用"

---

### 2.2 出库管理

#### 2.2.1 查询订单
**截图位置**：暂无

**操作步骤**：
1. 点击左侧菜单"出库管理"
2. 在订单号输入框中输入订单号
3. 点击"查询订单"按钮
4. 系统展示订单商品明细

#### 2.2.2 生成出库单
**截图位置**：暂无

**操作步骤**：
1. 查询订单后，确认订单明细
2. 点击"生成出库单"按钮
3. 系统根据先进先出策略推荐拣选货位
4. 确认后创建出库任务

#### 2.2.3 完成出库
**截图位置**：暂无

**操作步骤**：
1. 在出库单列表中，找到待出库任务
2. 点击"完成出库"按钮
3. 系统自动扣减库存
4. 若货位库存为零，自动标记为空闲

## 附录：常见问题

### 登录问题
**Q：忘记密码怎么办？**
A：请联系系统管理员重置密码。

**Q：登录提示"用户不存在"？**
A：确认工号/手机号输入正确，或联系管理员确认账户状态。

### 操作问题
**Q：入库时提示货位不可用？**
A：请确认货位状态为"空闲"，或选择其他空闲货位。

**Q：出库时提示库存不足？**
A：检查库存是否足够，或存在其他出库任务占用库存。

**Q：任务分配给操作员后，对方没收到？**
A：检查WebSocket连接是否正常，或让操作员刷新页面。

### 系统问题
**Q：页面显示异常？**
A：清除浏览器缓存后重试，或更换浏览器访问。

**Q：数据备份失败？**
A：检查服务器/backup目录是否可写，确认MySQL服务正常运行。
