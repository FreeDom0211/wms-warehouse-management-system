## 1. 环境要求

### 1.1 硬件要求
| 配置项 | 最低要求 | 推荐配置 |
|--------|----------|----------|
| CPU | 2核 | 4核 |
| 内存 | 4GB | 8GB |
| 磁盘 | 50GB | 100GB |

### 1.2 软件要求
| 软件 | 版本要求 | 说明 |
|------|----------|------|
| JDK | 1.8u281+ | 推荐使用JDK 8u311 |

- CentOS 7.x / 8.x
- Ubuntu 18.04 / 20.04 / 22.04
- Windows Server 2016+（开发测试环境）

## 2. 服务器准备

### 2.1 安装JDK 8

#### CentOS/RHEL
```bash
# 安装OpenJDK 8
sudo yum install -y java-1.8.0-openjdk java-1.8.0-openjdk-devel

# 验证安装
java -version

# 设置环境变量
echo 'export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk' >> ~/.bashrc
echo 'export PATH=$JAVA_HOME/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

#### Ubuntu/Debian
```bash
# 安装OpenJDK 8
sudo apt update
sudo apt install -y openjdk-8-jdk

# 验证安装
java -version
```

#### Windows
1. 下载JDK 8安装包：https://www.oracle.com/java/technologies/downloads/#java8
2. 双击安装，按照提示完成安装
3. 设置环境变量：
   - 新建 `JAVA_HOME` = `C:\Program Files\Java\jdk1.8.0_xxx`
   - 编辑 `PATH` 添加 `%JAVA_HOME%\bin`

### 2.2 安装MySQL 8.0

#### CentOS 7
```bash
# 添加MySQL Yum仓库
sudo yum install -y https://dev.mysql.com/get/mysql80-community-release-el7-7.noarch.rpm

# 安装MySQL Server
sudo yum install -y mysql-community-server

# 启动并设置开机启动
sudo systemctl start mysqld
sudo systemctl enable mysqld

# 获取临时密码
sudo grep 'temporary password' /var/log/mysqld.log

# 登录MySQL
mysql -u root -p

# 修改root密码
ALTER USER 'root'@'localhost' IDENTIFIED BY 'YourPassword123!';
```

#### Ubuntu
```bash
# 安装MySQL Server
sudo apt update
sudo apt install -y mysql-server

# 启动服务
sudo systemctl start mysql
sudo systemctl enable mysql

# 安全初始化
sudo mysql_secure_installation
```

### 2.3 安装Redis

#### CentOS/RHEL
```bash
# 安装Redis
sudo yum install -y redis

# 启动并设置开机启动
sudo systemctl start redis
sudo systemctl enable redis

# 验证Redis运行
redis-cli ping
```

#### Ubuntu
```bash
# 安装Redis
sudo apt update
sudo apt install -y redis-server

# 启动服务
sudo systemctl start redis-server
sudo systemctl enable redis-server

# 验证Redis运行
redis-cli ping
```

### 2.4 安装Maven

#### CentOS/RHEL
```bash
# 下载Maven
wget https://archive.apache.org/dist/maven/maven-3/3.8.6/binaries/apache-maven-3.8.6-bin.tar.gz
sudo tar -xzf apache-maven-3.8.6-bin.tar.gz -C /opt
sudo ln -s /opt/apache-maven-3.8.6 /opt/maven

# 设置环境变量
echo 'export MAVEN_HOME=/opt/maven' >> ~/.bashrc
echo 'export PATH=$MAVEN_HOME/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# 验证安装
mvn -version
```

## 3. 数据库初始化

### 3.1 创建数据库
```bash
# 登录MySQL
mysql -u root -p

# 创建数据库
CREATE DATABASE IF NOT EXISTS jd_wms DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 创建专用用户（可选）
CREATE USER 'wms_user'@'%' IDENTIFIED BY 'WmsPassword123!';
GRANT ALL PRIVILEGES ON jd_wms.* TO 'wms_user'@'%';
FLUSH PRIVILEGES;
```

### 3.2 执行初始化脚本
```bash
# 进入项目SQL目录
cd /path/to/project/sql

# 执行初始化脚本
mysql -u root -p jd_wms < init.sql

# 如果需要导入其他数据
mysql -u root -p jd_wms < init_manager.sql
mysql -u root -p jd_wms < init_operation_log.sql
```

### 3.3 验证数据库
```sql
-- 登录MySQL
mysql -u root -p jd_wms

-- 查看表
SHOW TABLES;

-- 查看用户表数据
SELECT id, work_no, name, role_code FROM user;
```

## 4. 项目构建

### 4.1 获取项目源码
```bash
# 克隆项目（如果有远程仓库）
git clone https://your-repo-url/jd-wms.git
cd jd-wms

# 或将项目文件复制到服务器
```

### 4.2 配置数据库连接
编辑 `wms-web/src/main/resources/application-prod.yml`：
```yaml
spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/jd_wms?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: YourPassword123!
  redis:
    host: localhost
    port: 6379
    password: # 如果设置了密码，在这里配置
```

### 4.3 构建项目
```bash
# 进入项目根目录
cd /path/to/jd-wms

# 清理并打包（跳过测试）
mvn clean package -DskipTests

# 或者完整打包（包括测试）
mvn clean package
```

构建成功后，在 `wms-web/target/` 目录下生成 `wms-web-1.0.0-SNAPSHOT.jar`。

### 4.4 创建备份目录
```bash
# 创建备份目录
sudo mkdir -p /backup
sudo chmod 777 /backup
```

## 5. 应用启动

### 5.1 启动命令

#### 开发环境启动
```bash
cd wms-web
mvn spring-boot:run
```

#### 生产环境启动（Jar包方式）
```bash
# 进入target目录
cd wms-web/target

# 启动应用（前台运行）
java -jar wms-web-1.0.0-SNAPSHOT.jar --spring.profiles.active=prod

# 启动应用（后台运行）
nohup java -jar wms-web-1.0.0-SNAPSHOT.jar --spring.profiles.active=prod > app.log 2>&1 &

# 查看启动日志
tail -f app.log

# 查看启动状态
ps -ef | grep wms-web
```

### 5.2 启动参数说明
| 参数 | 说明 |
|------|------|
| --spring.profiles.active=prod | 指定生产环境配置 |
| -Xms512m -Xmx1024m | JVM内存配置 |
| -XX:+UseG1GC | 使用G1垃圾收集器 |
| --server.port=8080 | 指定服务端口 |

### 5.3 创建系统服务（Systemd）
创建服务文件：
```bash
sudo vim /etc/systemd/system/wms.service
```

内容：
```ini
[Unit]
Description=JD WMS Application
After=network.target mysql.service redis.service

[Service]
Type=simple
User=root
WorkingDirectory=/path/to/jd-wms/wms-web/target
ExecStart=/usr/bin/java -Xms512m -Xmx1024m -jar wms-web-1.0.0-SNAPSHOT.jar --spring.profiles.active=prod
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

注册并启动服务：
```bash
# 重新加载systemd配置
sudo systemctl daemon-reload

# 设置开机启动
sudo systemctl enable wms

# 启动服务
sudo systemctl start wms

# 查看服务状态
sudo systemctl status wms

# 重启服务
sudo systemctl restart wms

# 停止服务
sudo systemctl stop wms
```

## 6. Nginx反向代理配置

### 6.1 安装Nginx

#### CentOS/RHEL
```bash
sudo yum install -y nginx

# 启动并设置开机启动
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### Ubuntu
```bash
sudo apt update
sudo apt install -y nginx

# 启动并设置开机启动
sudo systemctl start nginx
sudo systemctl enable nginx
```

### 6.2 配置反向代理
```bash
sudo vim /etc/nginx/conf.d/wms.conf
```

内容：
```nginx
upstream wms_backend {
    server 127.0.0.1:8080;
    keepalive 32;
}

server {
    listen 80;
    server_name your-domain.com;  # 替换为实际域名或IP

    # 强制跳转HTTPS（可选）
    # return 301 https://$server_name$request_uri;

    client_max_body_size 100M;

    location / {
        root /path/to/jd-wms/wms-web/src/main/resources/static;
        index index.html;

        # API请求代理到后端
        location /api/ {
            proxy_pass http://wms_backend;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
            proxy_set_header X-Forwarded-Proto $scheme;

            # 超时设置
            proxy_connect_timeout 60s;
            proxy_send_timeout 60s;
            proxy_read_timeout 60s;
        }

        # WebSocket支持
        location /ws/ {
            proxy_pass http://wms_backend;
            proxy_http_version 1.1;
            proxy_set_header Upgrade $http_upgrade;
            proxy_set_header Connection "upgrade";
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
        }
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        root /path/to/jd-wms/wms-web/src/main/resources/static;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    # 日志配置
    access_log /var/log/nginx/wms_access.log;
    error_log /var/log/nginx/wms_error.log;
}
```

### 6.3 配置HTTPS（可选）
```nginx
server {
    listen 443 ssl http2;
    server_name your-domain.com;

    ssl_certificate /etc/nginx/ssl/cert.pem;
    ssl_certificate_key /etc/nginx/ssl/key.pem;
    ssl_session_timeout 1d;
    ssl_session_cache shared:SSL:50m;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256;
    ssl_prefer_server_ciphers off;

    # 其他配置同上...
}
```

### 6.4 验证配置并重启
```bash
# 检查配置语法
sudo nginx -t

# 重载配置
sudo systemctl reload nginx

# 重启Nginx
sudo systemctl restart nginx
```

## 8. 常见问题处理

### 8.1 应用启动失败
**问题**：Java应用无法启动

**排查步骤**：
```bash
# 查看启动日志
tail -f app.log

# 检查端口占用
netstat -tlnp | grep 8080

# 检查JDK版本
java -version

# 检查数据库连接
mysql -u root -p -e "SELECT 1"
```

### 8.2 数据库连接失败
**问题**：应用无法连接MySQL

**排查步骤**：
```bash
# 检查MySQL服务
sudo systemctl status mysql

# 检查端口
netstat -tlnp | grep 3306

# 测试连接
mysql -u root -p jd_wms -e "SELECT 1"
```

**解决方案**：
```yaml
# 修改application-prod.yml中的数据库配置
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/jd_wms?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true&connectTimeout=30000&socketTimeout=60000
```

### 8.3 Redis连接失败
**问题**：应用无法连接Redis

**排查步骤**：
```bash
# 检查Redis服务
sudo systemctl status redis

# 测试连接
redis-cli ping

# 检查Redis配置
cat /etc/redis/redis.conf | grep bind
```

### 8.4 前端页面加载失败
**问题**：Nginx代理后页面无法访问

**排查步骤**：
```bash
# 检查静态文件路径
ls -la /path/to/jd-wms/wms-web/src/main/resources/static/

# 检查Nginx日志
tail -f /var/log/nginx/wms_error.log

# 检查SELinux（CentOS）
getsebool httpd_can_network_connect
setsebool -P httpd_can_network_connect 1
```

### 8.5 内存不足
**问题**：应用运行缓慢或被杀死

**排查步骤**：
```bash
# 查看内存使用
free -h

# 查看OOM日志
dmesg | grep -i oom
```

**解决方案**：
```bash
# 增加交换分区
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile

# 或优化JVM参数
java -Xms256m -Xmx512m -jar wms-web-1.0.0-SNAPSHOT.jar
```
